## Getting the code

1. Clone this repository

```
git clone https://github.com/jespertheend/fake-imports.git
cd fake-imports
```

2. [Install Deno](https://deno.land/#installation)

## Running tests

```
deno test -A --no-check=remote
```

## Code style

Before submitting a PR, make sure to run

```
deno fmt
deno lint
```
